## Common Tasks

### Entering Emission Data

1. Go to **Emission Records** ‚Üí **New Entry**
2. Fill required fields: Date, Facility, Scope, Source, Activity Data, Factor
3. Click **Save**

**Quick Entry (Multiple Records):**
1. Go to **Emission Records** ‚Üí **Quick Entry**
2. Click **Add Row** for each record
3. Fill fields and click **Save All**

### Generating a Report

1. Go to **Reports** ‚Üí **GHG Protocol Report**
2. Select date range and scope
3. Click **Generate Report**
4. Export as PDF or Excel

### Managing Suppliers

1. Go to **Scope 3** ‚Üí **Suppliers**
2. Click **Add Supplier**
3. Fill supplier information
4. Save

### Setting a Target

1. Go to **Targets** ‚Üí **Create Target**
2. Enter baseline and target values
3. Select scope and year
4. Save

### Importing Data

1. Go to **Emission Records** ‚Üí **Import**
2. Download sample template
3. Fill template with your data
4. Upload file
5. Review and confirm import
